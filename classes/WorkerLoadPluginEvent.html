Class WorkerLoadPluginEvent

Emitted when a plugin is loaded into the server somewhere, either in the worker itself globally, or in a specific room. Use WorkerLoadPluginEvent.isRoomPlugin to find out programmatically what kind of plugin has been loaded.

If you listen for this event in your plugin, you should probably set the order in your PluginMetadata to "first".

This is called before the Plugin.onPluginLoad lifecycle method.
