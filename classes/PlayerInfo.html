Class PlayerInfo<RoomType>

Represents information about a player, including any cosmetics they have equipped, whether they are dead or the impostor, their role and whatever tasks they have.

roleType

roleType: typeof BaseRole
  • Defined in node_modules/@skeldjs/core/dist/lib/misc/PlayerInfo.d.ts:82

Which player this role is. Note that this is not their actual instance of this role, see PlayerData.role, but is instead the class used to create the instance, and holds metadata about the role, such as the team it is for and whether it's a role for ghosts or not.

You can use this to know what class PlayerData.role will be an instance of.

example
if (player.playerInfo.roleType === ImpostorRole) {
  player.role instanceof ImpostorRole // true
}
